# 快捕 Flutter 开发文档

## 一、项目概述

### 1.1 项目简介
快捕是一个跨平台小说阅读器应用，支持 Windows 和 Android 平台。基于 Flutter 框架开发，提供书源管理、小说搜索、在线阅读、离线下载、朗读等功能。

### 1.2 技术栈
- **框架**: Flutter 3.x
- **语言**: Dart
- **数据库**: SQLite
- **网络请求**: Dart http 包
- **TTS**: 平台原生 TTS（Windows SAPI / Android TTS）
- **HTML解析**: html 包

## 三、关键功能实现

### 3.1 书源解析

书源使用URL模板规则，通过占位符匹配和替换来构建请求URL：

**基本格式**:
```
网站名称=书源名称
网站网址=https://example.com/
网站编码=UTF-8
简介页网址规则=https://example.com/(书类)/(书号)/
目录页网址规则=https://example.com/(书类)/(书号)/all.html
章节页网址规则=https://example.com/(书类)/(书号)/(章号).html
目录章节排序方式=按HTML先后排列
搜索网址=https://example.com/search?key={key}
搜索类型=GET
分类排行=分类名::URL模板&&分类名2::URL模板2
搜索状态=1
分类状态=1
```

**占位符说明**:
| 占位符 | 说明 |
|--------|------|
| `{key}` | 搜索关键词 |
| `{page}` | 页码 |
| `(书类)` | 书籍分类标识 |
| `(书号)` | 书籍ID |
| `(章号)` | 章节ID |

**实际示例**:
```
网站名称=壁落小说
网站网址=https://www.biquluo.info/
网站编码=UTF-8
简介页网址规则=https://www.biquluo.info/(书类)/(书号)/
目录页网址规则=https://www.biquluo.info/(书类)/(书号)/
章节页网址规则=https://www.biquluo.info/(书类)/(书号)/(章号).html
目录章节排序方式=按HTML先后排列
搜索网址=https://www.biquluo.info/search.html?name={key}
搜索类型=GET
分类排行=玄幻小说::https://www.biquluo.info/xuanhuan_{page}.html&&仙侠小说::https://www.biquluo.info/xianxia_{page}.html
搜索状态=1
分类状态=1
```

**POST搜索示例**:
```
搜索网址=https://example.com/search,{keyword={key}&t=1}
搜索类型=POST
```

### 3.5 编码处理

**支持的编码**:
- UTF-8（默认）
- GBK / GB2312（使用 fast_gbk 包）

**POST请求编码**:
对于GBK网站，POST请求体需要进行特殊处理：
```dart
// 将参数值进行GBK编码后再URL编码
final gbkBytes = gbk.encode(value);
final encodedValue = gbkBytes.map((b) => '%${b.toRadixString(16).padLeft(2, '0')}').join();
```

**编码检测优先级**:
1. 用户指定编码
2. HTTP响应头 Content-Type
3. HTML meta 标签
4. 默认UTF-8

4. **朗读异常**: Windows检查SAPI，Android检查TTS引擎
5. **搜索无结果**: 
   - 检查网站编码是否正确
   - 检查搜索URL是否有效
   - 检查简介页网址规则是否匹配
6. **乱码问题**: 
   - 检查网站编码设置
   - GBK网站需要正确设置编码

### 9.2 Book 数据结构
```dart
class Book {
  int? id;                     // 数据库ID
  String title;                // 书名
  String author;               // 作者
  String? coverUrl;            // 封面URL
  String? description;         // 简介
  String? lastReadChapter;     // 最后阅读章节名
  int lastReadChapterIndex;    // 最后阅读章节索引
  int totalChapters;           // 总章节数
  String? sourceUrl;           // 目录页URL
  String? sourceName;          // 书源名称
  DateTime? lastReadTime;      // 最后阅读时间（用于排序）
  DateTime? addedTime;         // 加入时间
  String? localPath;           // 本地路径
  int readProgress;            // 阅读进度百分比
  bool isFavorite;             // 是否收藏
}
```

**书架排序规则**:
- 按 `lastReadTime` 降序排列
- 新加入的书籍 `lastReadTime` 设为当前时间，排在前面
- 阅读时更新 `lastReadTime`，自动排到最前面
